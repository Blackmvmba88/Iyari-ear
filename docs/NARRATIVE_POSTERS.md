## 📐 Especificaciones Técnicas

```
Tipografía: Monospace (Courier, Consolas, Monaco)
Ancho: 59 caracteres (estándar terminal 80 cols con margen)
Alto: Variable según contenido
Bordes: Box-drawing characters (╔═╗║╚╝)
Formato: Plain text preservado en Markdown code blocks
```

---

## 💡 Filosofía de Diseño

**¿Por qué ASCII art y no imágenes?**

1. **Accesibilidad**: Screen readers pueden leer el contenido
2. **Portabilidad**: Funciona en cualquier dispositivo sin dependencias
3. **Versionamiento**: Git puede hacer diff y mostrar cambios
4. **Minimalismo**: Coherente con la filosofía del proyecto
5. **Nostalgia técnica**: Honra las raíces de la computación
