# Matrix Rain Demo

Demo visual estilo Matrix con efectos de lluvia de caracteres.

## Características

- Efectos de lluvia de caracteres estilo Matrix
- Modo arcoíris para colores dinámicos
- Visualizador de audio reactivo
- Controles personalizables:
  - Color de caracteres
  - Tamaño de fuente
  - Velocidad de animación
  - Densidad de lluvia
  - Set de caracteres (Katakana, Latino, Números, Binario)
- Modo pantalla completa

## Cómo usar

1. Abre `index.html` en tu navegador
2. Ajusta los controles según tus preferencias
3. Haz clic en "Pantalla Completa" para una experiencia inmersiva
4. Usa "Ocultar Controles" para ver solo la animación

## Archivos

- `index.html`: Página HTML del demo
- `style.css`: Estilos para el canvas y controles
- `matrix.js`: Lógica de animación y efectos

## Nota

Este es un demo visual independiente del proyecto principal Iyari-ear.
No es necesario para la funcionalidad de subtítulos en tiempo real.
