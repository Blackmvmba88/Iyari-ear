# 🎤 v0.1 — First Listening

> **El primer paso para que nadie se quede fuera de la conversación.**

---

## 🌟 ¿Qué es Iyari-ear?

Iyari-ear es una aplicación web que genera **subtítulos en tiempo real** para conversaciones cara a cara. 

Un celular sobre la mesa. Alguien habla. Las palabras aparecen.

Es una herramienta de accesibilidad pensada para personas con dificultades auditivas, diseñada con empatía y sin fricción técnica.

---

## 💡 ¿Qué problema humano resuelve?

Imagina estar en:
- Una cafetería ruidosa con amigos
- Una consulta médica importante
- Una comida familiar

Para alguien con pérdida auditiva, estas situaciones pueden ser:
- Agotadoras (pedir que repitan constantemente)
- Aislantes (perderse partes de la conversación)
- Frustrantes (no poder participar naturalmente)

**Iyari-ear permite:**
- Seguir conversaciones sin interrumpir
- Participar sin esfuerzo extra
- Conectar sin barreras

---

## 🚫 ¿Qué NO hace Iyari-ear?

Esta herramienta está diseñada con límites éticos claros:

❌ **No graba audio**
❌ **No almacena conversaciones**
❌ **No es un sistema de vigilancia**
❌ **No reemplaza intérpretes de lengua de señas**
❌ **No es una herramienta forense o legal**

### ¿Por qué estas limitaciones son importantes?

Porque Iyari-ear es una herramienta de **apoyo**, no de **control**.

La tecnología debe ayudar a escuchar sin invadir.
Apoyar sin vigilar.
Mostrar palabras, no juzgar voces.

---

## 🛠️ ¿Cómo funciona?

```
Micrófono → Audio → Servidor → API de voz → Texto → Pantalla
```

En tiempo real. Sin fricción.

**Tecnologías usadas:**
- Frontend: HTML, CSS, JavaScript (Web Audio API)
- Backend: Python, FastAPI, WebSockets
- Reconocimiento de voz: Google Speech Recognition API
