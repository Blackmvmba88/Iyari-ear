## ⏱️ 15 Minutos para Hacer el Repo Visible

### Paso 1: GitHub Settings (5 min) ⚙️

1. Ve a tu repo en GitHub
2. Click en ⚙️ al lado de "About"
3. Pega esto:

**Description:**
```
Subtítulos en tiempo real para conversaciones cara a cara. Accesibilidad simple, ética y humana.
```

**Topics (uno por línea para copiar fácil):**
```
accessibility
speech-to-text
realtime
assistive-technology
ethical-tech
web-audio
subtitles
hearing-support
```

O todos juntos separados por espacios:
```
accessibility speech-to-text realtime assistive-technology ethical-tech web-audio subtitles hearing-support
```

4. Save ✅

👉 **[Guía detallada](./PASO_1_CONFIGURACION_GITHUB.md)**

---

### Paso 2: Primer Release (10 min) 🏷️

1. Ve a GitHub → tu repo → "Releases" (lado derecho)
2. Click "Create a new release"
3. Llena:
   - **Tag:** `v0.1`
   - **Title:** `v0.1 — First Listening`
   - **Description:** Abre [RELEASE_NOTES_v0.1.md](./RELEASE_NOTES_v0.1.md) y copia TODO el contenido
4. Marca "Set as the latest release"
5. Click "Publish release" ✅

👉 **[Ver contenido del release](./RELEASE_NOTES_v0.1.md)**

---

## ✨ ¡Listo! Tu repo ya tiene identidad

Con solo esos 2 pasos:
- ✅ Aparece en búsquedas
- ✅ Se entiende en 3 segundos
- ✅ Tiene release oficial
- ✅ Ya no parece experimental
