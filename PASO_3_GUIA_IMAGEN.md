# 📸 Paso 3: Guía para Crear la Imagen Demostrativa

## 🎯 Objetivo

Crear una imagen simple y efectiva que muestre **qué es Iyari-ear** en 3 segundos, sin necesidad de leer.

---

## 🖼️ Descripción de la Imagen

### Elementos principales:
1. **Un celular moderno** (smartphone) visto desde arriba o ligeramente inclinado
2. **Sobre una mesa** con superficie neutra (madera clara, gris, o blanco)
3. **Pantalla del celular encendida** mostrando texto en tiempo real
4. **Texto grande y legible** en la pantalla (subtítulos)
5. **Fondo minimalista** sin distracciones

### Estilo visual:
- **Limpio y profesional**, no caricaturesco
- **Colores neutros** con un acento de color (#00ff9f verde neón para el texto)
- **Iluminación suave y natural**
- **Enfoque en la pantalla del celular**

---

## 🤖 Prompt para Generador de Imágenes AI

### Opción 1: Prompt detallado (DALL-E, Midjourney, Stable Diffusion)

```
A modern smartphone laying flat on a light wooden table, viewed from directly above. The phone screen is displaying large, clear subtitle text in bright neon green (#00ff9f) on a dark background. The text on screen reads "Hola, ¿cómo estás?" and "Muy bien, gracias." in a clean sans-serif font. Minimalist composition, soft natural lighting, neutral background, professional photography style, shallow depth of field with the phone in sharp focus. Photorealistic, 4K quality, clean and accessible design aesthetic.
```

### Opción 2: Prompt simplificado

```
Smartphone on table showing real-time subtitles on screen, text in bright green, dark background, minimalist, top view, photorealistic
```

### Opción 3: Prompt en español (para generadores que lo soporten)

```
Un celular moderno sobre una mesa de madera clara, vista desde arriba. La pantalla muestra subtítulos en tiempo real en verde neón sobre fondo oscuro. El texto dice "¿Quieres café?" y "Sí, por favor." Estilo minimalista, iluminación natural suave, fotografía profesional, alta calidad.
```

---

## 📐 Especificaciones Técnicas

- **Resolución recomendada:** 1200x675px (16:9) o 800x450px
- **Formato:** PNG o JPG
- **Peso:** Menos de 500KB (optimizado para web)
- **Orientación:** Horizontal (landscape)

---

## 🎨 Elementos de Texto en Pantalla

El texto que aparece en la pantalla del celular debe ser:

**Opción A (conversación casual):**
```
— Hola, ¿cómo estás?
— Muy bien, gracias.
— ¿Quieres café?
```

**Opción B (más emocional):**
```
— Te quiero mucho
— Yo también
```

**Opción C (contexto médico):**
```
Doctor: ¿Cómo te sientes hoy?
Paciente: Mucho mejor.
```

---

## 🛠️ Herramientas Sugeridas

### Generadores de Imágenes AI:
1. **DALL-E 3** (OpenAI) - Mejor calidad fotorrealista
2. **Midjourney** - Excelente para composiciones artísticas
3. **Stable Diffusion** - Open source, control total
4. **Leonardo.ai** - Buena alternativa gratuita

### Editores de Imágenes:
- **Figma** - Para mockups de UI
- **Canva** - Para composiciones simples
- **Photoshop** - Para edición profesional

---

## ✅ Checklist de Calidad

Antes de usar la imagen, verifica:

- [ ] La pantalla del celular es clara y legible
- [ ] El texto tiene suficiente contraste
- [ ] No hay elementos distractores en el fondo
- [ ] La imagen se ve bien tanto en escritorio como móvil
- [ ] El tamaño del archivo es menor a 500KB
- [ ] La imagen transmite "accesibilidad" y "simplicidad"

---

## 📍 Dónde Colocar la Imagen

Una vez generada, la imagen debe:

1. **Subirse al repositorio** en una carpeta llamada `assets/` o `images/`
2. **Actualizarse en el README.md** reemplazando el placeholder actual
3. **Usarse en redes sociales** al compartir el proyecto

### Código para el README:

```markdown
<div align="center">

![Iyari-ear Demo](./assets/iyari-ear-demo.png)

*Subtítulos en tiempo real en tu celular. Sin fricción. Solo conexión.*

</div>
```

## 💡 Consejos Finales

### Lo que SÍ debe transmitir la imagen:
✅ Accesibilidad
✅ Simplicidad
✅ Claridad
✅ Tecnología al servicio del humano

### Lo que NO debe transmitir:
❌ Complejidad técnica
❌ Vigilancia o control
❌ Diseño corporativo frío
❌ Marketing agresivo
