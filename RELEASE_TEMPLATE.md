## ✨ Características Principales

### 🎤 Sistema de Subtítulos en Tiempo Real
- Transcripción en vivo para conversaciones cara a cara
- Soporte multiidioma (Español e Inglés)
- PWA instalable - funciona como app nativa
- Interfaz accesible con alto contraste
- Indicador visual de detección de voz

### 🔧 Sistema de Diagnóstico Electrónico
- 3 Capas de diagnóstico (Localización → Causa → Consecuencia)
- Análisis multi-foto con razonamiento causal
- Reportes profesionales completos
- Estilos: Técnico, Ingeniero, Forense

### 🎬 Optimizador de Subtítulos
- Validación y optimización de SRT, VTT, ASS
- Plugin VLC para integración directa
- CLI potente para procesamiento por lotes
- Interfaz web drag-and-drop

### 🌐 Multiplataforma
- Windows (ejecutable .exe)
- Linux (script de instalación + systemd)
- macOS (Homebrew)
- Android (Termux + PWA)
- Web (acceso universal)

---

## 🚀 Instalación Rápida

### PWA (Recomendado)
```bash
pip install -r requirements.txt
python main.py
# Abre http://localhost:8000
# Click "Instalar" en la barra del navegador
```

### CLI
```bash
pip install -e .
iyari-ear doctor     # Verifica sistema
iyari-ear start      # Inicia servidor
```

## 🚫 Limitaciones Conocidas

Estas limitaciones son **parte del diseño ético**:

- ⚠️ Requiere conexión a internet
- ⚠️ Precisión variable en ambientes muy ruidosos
- ⚠️ Acentos marcados pueden afectar transcripción
- ⚠️ No diseñada para transcripción legal/forense

**Estas no son bugs. Son límites intencionales.**

## Instrucciones para Publicar el Release

1. **Ve a GitHub**:
   - Navega a: https://github.com/Blackmvmba88/Iyari-ear
   - Click en "Releases" (lateral derecho)

2. **Crea Nuevo Release**:
   - Click "Draft a new release"
   - Tag: `v1.0.0`
   - Título: `v1.0 — "Para que nadie quede fuera de la conversación"`

3. **Pega el Contenido**:
   - Copia el contenido de arriba en la descripción
   - Ajusta cualquier link si es necesario

4. **Marca como Release Principal**:
   - ✅ Set as the latest release
   - ✅ Create a discussion for this release (opcional)

5. **Publica**:
   - Click "Publish release"

6. **Comparte**:
   - Twitter/X: Anuncia el release con #IyariEar
   - LinkedIn: Post sobre el lanzamiento
   - Dev.to / Hashnode: Artículo sobre la historia
